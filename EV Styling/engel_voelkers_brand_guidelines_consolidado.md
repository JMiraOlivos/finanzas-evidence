# Guía consolidada de marca — Engel & Völkers

> **Propósito:** Este documento resume en un solo archivo las directrices de identidad visual, color, tipografía, composición, UI, voz, tono, experiencia y producción impresa de Engel & Völkers. Está redactado para funcionar como guía operativa y también como set de instrucciones para una IA generativa o asistente de diseño/contenido.

---

## 1. Esencia de marca

Engel & Völkers debe percibirse como una marca premium, sofisticada, segura y humana. La experiencia debe combinar lujo, claridad y atención al detalle, evitando excesos ornamentales o decisiones visuales que reduzcan la sensación de confianza.

### Principios de experiencia

Toda pieza o interacción debe responder a cuatro principios:

| Principio | Cómo se expresa |
|---|---|
| **Atenta / Attentive** | Escucha activa, empatía genuina, atención al detalle y capacidad de transformar lo ordinario en algo extraordinario. |
| **Analítica / Exigente / Discerning** | Calidad superior, refinamiento, precisión y valor real en cada interacción. |
| **Dinámica / Dynamic** | Evolución continua, agilidad, liderazgo y adaptación sin perder consistencia. |
| **Segura de sí misma / Confident** | Presencia sólida, claridad, decisión y capacidad de destacar con elegancia. |

### Regla general para IA

Cuando generes una pieza para Engel & Völkers, prioriza siempre:

1. **Espacio en blanco generoso.**
2. **Composición limpia y estructurada.**
3. **Sensación premium y editorial.**
4. **Uso moderado del rojo.**
5. **Fotografía real antes que ilustración.**
6. **Mensajes breves, claros y seguros.**
7. **Consistencia absoluta entre puntos de contacto físicos y digitales.**

## 2. Paleta de colores

La paleta de Engel & Völkers se organiza en tres niveles: primaria, secundaria y funcional.

### 2.1 Paleta primaria

La paleta primaria tiene precedencia sobre todas las demás. Define la identidad esencial de la marca.

| Color | HEX | RGB | CMYK | PMS / RAL | Uso principal |
|---|---:|---:|---:|---|---|
| **White** | `#FFFFFF` | 255, 255, 255 | 0, 0, 0, 0 | — | Fondo principal. Debe crear una apariencia limpia, lujosa y sofisticada. |
| **Brand Black / Dark Gray** | `#303030`* | 48, 48, 48 | 0, 0, 0, 95 | PMS 447 C / RAL 9005 | Wordmark, textos, fondos oscuros audaces, iconos negros. |
| **Brand Red** | `#E60000` | 230, 0, 0 | 0, 100, 100, 0 | PMS 3546 C / RAL 3020 | Ampersand, líneas rojas, acentos, elementos destacados y recursos de identidad. |

\* **Nota de consistencia:** el resumen de color cargado indica `HEX #000D0D` junto con `RGB 48,48,48`. Como esos valores no son equivalentes, para producción digital se recomienda validar el valor oficial en el portal de marca. En este documento se usa `#303030` porque corresponde al RGB indicado.

### Uso de la paleta primaria

- Usa **blanco** como fondo dominante.
- Usa **Brand Black / Dark Gray** para textos, wordmark, iconos negros y aplicaciones sobrias.
- Usa **Brand Red** de forma contenida, como acento de identidad. Nunca debe dominar la composición.
- No uses grandes masas de rojo.
- No uses la paleta primaria para infografías cuando la guía indique usar la paleta funcional.

## 3. Tipografía

Engel & Völkers usa dos tipografías propias:

| Fuente | Rol | Uso |
|---|---|---|
| **Engel Voelkers Head** | Serif display | Titulares, hero blocks, mensajes principales. |
| **Engel Voelkers Text** | Sans serif secundaria | Subtítulos, cuerpo de texto, captions, footers, especificaciones y bloques informativos. |

### 3.1 Engel Voelkers Head

- Usa esta fuente para titulares y mensajes principales.
- Debe sentirse sofisticada, editorial y asociada a la marca.
- Los titulares deben ser breves, limpios y con mucho aire visual.
- Preferiblemente centrados.
- Interlineado recomendado para titulares: **130%**.
- Kerning: **Optical**.
- Tracking: **0**.
- No apliques tracking adicional a esta fuente serif.

### 3.2 Engel Voelkers Text

- Usa esta fuente para subtítulos, textos largos, información funcional y especificaciones.
- Para subtítulos: usar mayúsculas y **tracking +100**.
- Para cuerpo de texto: tracking **0**, kerning **Optical** y uso de punto final.
- Para textos de más de 6 líneas: alineación izquierda o justificada.
- Para información listada o specs: alineación izquierda dentro de la retícula, con separación clara entre título y dato.

### 3.3 Interlineado

| Tamaño de texto | Leading |
|---|---:|
| Mayor a 7 pt | 150% del tamaño de la tipografía |
| 7 pt o menor | 140% del tamaño de la tipografía |

### 3.4 Fuentes fallback

Cuando no sea posible usar las fuentes oficiales, por ejemplo en Microsoft Office o Google Workspace:

| Uso | Fallback |
|---|---|
| Titulares | Georgia |
| Texto / UI / cuerpo | Arial |

### 3.5 No hacer en tipografía

- No uses **Engel Voelkers Text** para titulares principales.
- No uses **Engel Voelkers Head** para cuerpo de texto.
- No escribas textos en rojo como estilo principal.
- No apliques tracking superior a +100 en subtítulos.
- No uses tracking +100 en minúsculas.
- No exageres el interlineado.
- No escribas “Engel & Völkers Head” en mayúsculas cuando te refieras a la fuente.

## 4. Sistema de cuadrícula y composición

El sistema de cuadrícula estructura la información, potencia el espacio en blanco y mantiene consistencia en formatos impresos y digitales.

### 4.1 Categorías de formato

| Categoría | Aplicación |
|---|---|
| **Formatos pequeños** | Impresos menores a DIN A6. Incluye excepciones como tarjetas de visita. |
| **Formatos fijos** | Impresos desde DIN A6 hasta DIN A4 y anuncios digitales estándar, como posts o stories. |
| **Formatos grandes** | Impresos DIN A3 o superiores y publicidad exterior / OOH. |

### 4.2 Orden de construcción del layout

Construye siempre en este orden:

1. Define **márgenes**.
2. Crea **columnas**.
3. Establece la **cuadrícula base / baseline grid**.

### 4.3 Columnas

- Las cuadrículas de columnas deben ser simétricas.
- Las columnas deben ajustarse perfectamente a los márgenes.
- Normalmente se usan **3 o 6 columnas**.
- Para mayor densidad de contenido, puede ampliarse a **9 o 12 columnas**.
- No uses una cantidad arbitraria de columnas.

### 4.4 Baseline grid

- Medida fija: **6.75 pt** para todas las comunicaciones impresas.
- Inicia exactamente desde el límite del margen superior.
- Sirve para controlar espaciado entre líneas, módulos de contenido e imágenes.
- La primera línea de texto de un bloque debe “colgar” del inicio de la baseline grid.

### 4.5 Reglas de layout

- Alinea imágenes con precisión a las guías de la cuadrícula.
- Acopla la primera línea de texto a la baseline grid.
- Evita layouts libres sin sistema.
- Usa el espacio en blanco como parte activa de la identidad.

## 5. Logotipo

El logotipo es un pilar central de la identidad visual. Debe comunicar sofisticación y posicionamiento premium.

### 5.1 Área de protección

- Mantén espacio libre alrededor del logo.
- El mínimo de seguridad está definido por la **altura de la letra “E”** del logotipo.
- No coloques textos, imágenes, patrones o elementos competitivos dentro de esa área.

### 5.2 Tamaños mínimos

| Medio | Tamaño mínimo |
|---|---:|
| Impresión | 25 mm de ancho |
| Digital | 166 px de ancho |

Si el espacio disponible es menor, usa el **isotipo / signet** en lugar del logotipo completo.

### 5.3 Escala

| Contexto | Regla |
|---|---|
| Sin contenido / logo aislado | Centrado en el espacio en blanco. En 16:9 ocupa 50% del ancho; en 1:1 o 9:16 ocupa 60%. |
| Con contenido, formatos grandes | 25% a 35% del ancho de la aplicación. |
| Con contenido, formatos pequeños | 30% a 45% del ancho de la aplicación. |

### 5.4 Posicionamiento

| Posición | Uso permitido |
|---|---|
| Centrado | Opción predeterminada para equilibrio visual. |
| Arriba centrado | Formatos horizontales o verticales con texto. Se alinea con el margen superior y se centra horizontalmente. |
| Abajo izquierda | Solo páginas internas de documentos corporativos o diapositivas. |
| Derecha | Solo imagen final de carruseles o galerías en redes sociales, como cierre de impacto. |

### 5.5 Variantes de color

| Variante | Fondo permitido |
|---|---|
| Logo en color | Solo sobre blanco puro. Excepción digital: grises muy claros con luminancia mínima de 92%. |
| Monocromático negativo / blanco | Fondos oscuros o imágenes oscuras. |
| Monocromático positivo / negro | Colores secundarios claros, materiales especiales o imágenes claras. |

### 5.6 No hacer con el logo

- No recrees el logo con tipografías comunes.
- No lo distorsiones, rotes ni deformes.
- No añadas sombras.
- No uses la versión en color sobre fondos oscuros, imágenes o colores secundarios.
- No invadas su área de seguridad.
- No uses versiones no aprobadas.

## 6. Isotipo / Signet

El isotipo “E&V” es una abreviatura del logotipo principal. Debe usarse únicamente cuando el espacio no permita usar la palabra completa.

> **Importante:** su uso está sujeto a que esté permitido explícitamente en el contrato de licencia correspondiente.

### 6.1 Tamaños mínimos

| Medio | Tamaño mínimo |
|---|---:|
| Impresión | 24 mm de ancho |
| Digital | 14 px de ancho |
| Excepción: lomo de revista / GG sleeve | 8 mm de ancho |

### 6.2 Área de protección

- Debe estar libre de textos, imágenes o elementos gráficos competitivos.
- El espacio mínimo se define por la **altura de la letra “E”**.
- En avatares y favicons, se puede aplicar la cuadrícula de iconos UI como excepción.

### 6.3 Posicionamiento

- Por regla general, debe colocarse completamente centrado.
- En avatares y favicons, adapta el encuadre al contenedor usando la UI icon grid.

### 6.4 No hacer con el signet

- No lo escribas o compongas con tipografías comunes.
- No uses la versión negativa blanca con el símbolo “&” en rojo.
- No lo uses en formatos grandes o anchos si hay espacio para el logo completo.
- No apliques la versión en color sobre fondos de colores secundarios.

## 7. Villa

La villa representa a Engel & Völkers como marca global y funciona como marca de calidad. Tiene un rol diferente al logotipo y aparece en comunicaciones residenciales, corporativas, colaboraciones y submarcas.

### 7.1 Versiones

- **Regular.**
- **Reduced / reducida**, para tamaños inferiores a los límites de la versión regular.

### 7.2 Tamaños mínimos de la versión regular

| Medio | Tamaño mínimo |
|---|---:|
| Impresión | 30 mm de ancho |
| Digital | 200 px de ancho |

Para tamaños menores, usa la versión reducida.

### 7.3 Proporción frente al logotipo

- Por norma general, la villa debe medir **50% del tamaño del logotipo**.
- En formatos gigantes / OOH puede aplicarse más pequeña para conservar espacio en blanco.
- En aplicaciones individuales, como tarjetas de visita, puede tener tamaño personalizado.

### 7.4 Funciones

| Función | Uso |
|---|---|
| **Sign off / cierre** | En piezas de una cara, se coloca en la parte inferior. En piezas de doble cara, en la contraportada inferior. |
| **Quality mark / marca de calidad** | Se usa aislada para reforzar lujo y excelencia. |

### 7.5 Área de protección y posicionamiento

- Mantén libre el área determinada por la **altura del tejado** de la villa.
- Puede ir centrada encima del bloque de contacto.
- Puede ir aislada y centrada como marca de calidad.
- Puede ir anclada al margen inferior si no hay bloque de contacto.
- La versión sólida solo se usa para troquelados en vinilo o serigrafía; no es una “villa negra” de uso general.

### 7.6 No hacer con la villa

- No la pongas sobre fondos de colores secundarios.
- No la pongas sobre imágenes de fondo.
- No la sitúes en la parte superior del diseño.
- No uses versión regular en tamaños pequeños ni versión reducida en formatos grandes.
- No la recortes horizontalmente.
- No modifiques el grosor de sus líneas.
- No le añadas iconos ni ilustraciones.
- No la vincules directamente al wordmark.

## 8. Línea roja

La línea roja es un recurso recurrente para ordenar, conectar y estructurar información, además de reforzar la identidad visual.

### 8.1 Orientación

| Tipo | Uso |
|---|---|
| Vertical | Entre columnas de texto o para conectar bloques superpuestos. Debe alinearse a la izquierda o centrarse con el gutter. |
| Horizontal | Elemento divisor entre contenidos. Debe centrarse respecto a los bloques que separa. |

### 8.2 Comportamientos

| Comportamiento | Uso | Reglas |
|---|---|---|
| **Connecting** | Conecta dos bloques de texto. | Siempre centrada. Longitud flexible. Debe calcularse con 2 unidades de baseline grid por encima y por debajo del texto. Longitud mínima: 8 unidades de baseline grid. |
| **Modular organisational** | Organiza módulos de texto. | Grupo de 1 a 3 líneas rojas. Nunca 4 o más. Todas deben tener la misma longitud. No deben ubicarse en bordes exteriores. |
| **Single organisational** | Divide o equilibra layouts abiertos. | Se sitúa a la izquierda de la caja de texto para anclar la lectura. Debe alinearse con el gutter. |
| **Separating** | Divide titulares y texto. | Solo si hay entre 6 y 12 unidades de baseline grid entre elementos. Si el espacio es mayor, usa línea vertical. Debe ocupar 1 o 2 columnas, nunca más. |

### 8.3 No hacer con la línea roja

- No uses más de un comportamiento de línea roja en una misma página.
- No uses línea de conexión si el espacio entre textos es muy pequeño; usa línea horizontal.
- No coloques líneas rojas en bordes exteriores de cajas de texto.
- No estires la línea roja a todo el ancho de la página.
- No combines la línea roja y la villa en la misma composición.
- No dejes una línea roja huérfana, sin contenido asociado.

## 9. Highlights

Los highlights son marcos rojos inspirados en la silueta de la villa. Sirven para realzar subtítulos, llamadas a la acción o indicadores gráficos.

### 9.1 Construcción

- El marco proviene de los contornos del tejado y el suelo de la villa.
- La línea superior y la inferior deben usarse siempre juntas.
- El texto debe ir en la sans serif de marca.
- Texto en mayúsculas.
- Tracking: **+100**.

### 9.2 Tipos

| Tipo | Color | Uso | Reglas |
|---|---|---|---|
| **Call to Action** | Líneas rojas + texto gris oscuro `CMYK 0,0,0,75 / RGB 100,100,100` | CTA breve. | Una sola línea. El texto se extiende más allá del marco. Máximo 30 caracteres. |
| **Signpost** | Líneas rojas + texto rojo | Indicador o señal visual. | Idealmente el texto queda dentro del ancho de las líneas. |

### 9.3 No hacer

- No uses frases de más de 5 palabras.
- No uses más de una línea de texto.
- No modifiques la separación vertical entre tejado y suelo.
- No alteres el grosor de las líneas rojas.
- No cambies proporciones entre tipografía y marco.
- No escales el highlight de forma desproporcionada.

## 10. Patrones

Los patrones se inspiran en las líneas de la villa y en el movimiento de luces y sombras. Deben transmitir lujo mediante una herramienta flexible y evolutiva.

### 10.1 Usos

| Medio | Uso |
|---|---|
| Impresión | Papelería corporativa y piezas de marketing. |
| Digital | Intros, transiciones, cierres y comunicaciones digitales. |
| Otros soportes | Merchandise, entornos/oficinas y rotulación de vehículos. |

### 10.2 Construcción

| Principio | Regla |
|---|---|
| **Grid** | Cuadrícula vertical. Tamaño estándar para formatos mayores a 428 px o 85 mm de ancho; tamaño mínimo para formatos menores. |
| **Matrix & texture** | La cuadrícula se enmascara en 3 áreas horizontales. La textura usa 3 grosores de línea: grueso, medio y fino, en gris oscuro. |
| **Shapes** | Solo 4 formas decorativas: líneas, olas, curvas y zig-zag. |
| **Behaviours** | Even: repetición uniforme. Gradual: repetición irregular y progresiva. |

### 10.3 Aplicación

| Modo | Regla |
|---|---|
| General | Ajustar a bordes laterales y parte superior o inferior. Nunca cubrir más de 50% del espacio vertical. |
| Front | Si la portada no lleva imagen, colocar el patrón abajo para apoyar el texto. |
| Back | Usar como cierre en la parte trasera solo cuando no se utilice la villa. |
| Wrapping | Adaptar a superficies tridimensionales como vehículos, productos o paredes. |

### 10.4 Reglas de layout

- Deja margen limpio a izquierda y derecha equivalente al espacio entre líneas de la cuadrícula.
- En digital, normalmente se posiciona abajo.
- En Instagram Stories y formatos pequeños se usa el patrón pequeño.
- En impresión puede ir arriba o abajo.
- En tarjetas de presentación y formatos menores a 85 mm, usa el tamaño pequeño.
- Para una sensación premium, se pueden usar acabados como embossing, foil stamping, grabado o vinilo.

### 10.5 No hacer

- No uses patrones sobre fondos de color.
- No recortes abruptamente las terminaciones.
- No uses negro puro; usa el gris oscuro de marca.
- No enmarques el patrón con bordes.
- No inventes formas geométricas fuera de las 4 oficiales.
- No cambies el color del patrón.

## 11. Iconografía

Los iconos adoptan el lenguaje visual del logotipo y de la villa. Deben ser personalizados, seguros, simples y legibles.

### 11.1 Color

| Icono | Fondo |
|---|---|
| Negro / gris oscuro `RGB 48,48,48 / CMYK 0,0,0,95` | Fondo blanco. |
| Blanco | Fondo negro u oscuro. |

No se deben colorear con tonos corporativos secundarios ni funcionales.

### 11.2 Categorías

| Categoría | Uso |
|---|---|
| **UI icons** | Información funcional en móvil, escritorio y web. |
| **Bullet point icons** | Apoyo visual a bloques de texto en digital e impresión. |
| **Illustrative icons** | Representación gráfica de tipos de propiedades. |
| **Motion icons** | Puntos de contacto funcionales y emocionales en presentaciones, landing pages y apps. |

### 11.3 Grids y construcción

| Tipo | Grid | Grosor | Padding |
|---|---:|---:|---:|
| UI icons | 24 × 24 px | 1 px | 2 px |
| Bullet point icons | 48 × 48 px | 2 px | 2 px |
| Illustrative icons | 120 × 120 px | Según sistema | 16 px |

### 11.4 Reglas de diseño

- Diseña al 100% de escala.
- Usa perspectiva frontal.
- Simplifica al máximo.
- Usa trazos sin contorno.
- Mantén grosores consistentes.
- Mantén distancia simétrica entre iconos, titulares y textos.
- En iconos ilustrativos:
  - Digital: ventanas y puertas en gris `#878787`; en hover o scroll pueden cambiar a beige claro `#F8F5F0`.
  - Impresión: ventanas solo en amarillo CMYK `3,4,7,0`.

### 11.5 Motion icons

- Usar transiciones fluidas como **fade & move** y **dissolve**.
- Deben aportar claridad, no distracción.

### 11.6 No hacer

- No colorees iconos con paletas secundarias o funcionales.
- No escales fuera de los rangos del sistema.
- No alteres grosores de línea de forma inconsistente.
- No sobrecomplique ni hagas iconos demasiado literales.

## 12. Ilustraciones

Las ilustraciones son un componente secundario del sistema visual. Combinan líneas vectoriales dibujadas a mano con colores pasteles claros basados en paletas oficiales.

### 12.1 Regla principal

**La fotografía real siempre tiene prioridad sobre la ilustración.** Las ilustraciones solo se usan cuando no sea posible representar la situación con una fotografía real.

### 12.2 Requisitos

- Deben ser creadas por un ilustrador profesional que replique el estilo definido.
- Deben recibir aprobación del equipo de marca corporativo antes de producción o publicación.
- Deben funcionar tanto en digital como en impresión.
- Deben aparecer sobre fondos claros, preferiblemente blanco puro.

### 12.3 Funciones

| Función | Uso |
|---|---|
| Dibujos aislados | Apoyar mensajes o instrucciones específicas. |
| Videos explicativos | Comunicar información de forma dinámica. |
| Infografías | Complementar gráficos informativos. |

### 12.4 Color

| Paleta | Descripción |
|---|---|
| **Brand** | Estilo dicromático: Brand Red como contraste frente al gris oscuro, líneas negras firmes hechas a mano y gris para suavidad/textura. |
| **Extended** | Colores secundarios y funcionales para un look más expresivo. Puede incluir tonos de piel sutiles para representar diversidad. |

### 12.5 Personas y lifestyle

- Representar un estilo de vida de lujo.
- Entornos, mobiliario y detalles deben reflejar sofisticación.
- Usar objetos de diseño, iluminación cuidada, texturas y materiales premium.
- Personas con postura digna, relajada y elegante.
- Ropa impecable / smart.
- Carisma tranquilo y alegre.
- Diversidad representada mediante atributos físicos, peinados y accesorios de forma sutil.

### 12.6 Escalado

- Tamaño digital recomendado: entre **500 px y 1920 px**.
- Escala la ilustración completa.
- No alteres el grosor de las líneas de forma independiente.

### 12.7 No hacer

- No reemplaces fotografías reales por ilustraciones en portadas o secciones principales.
- No mezcles fotografía e ilustración en una misma composición.
- No uses ilustraciones como header o key visual de campaña.
- No uses fondos oscuros.
- No añadas colores fuera de paleta.
- No uses ilustraciones monocromáticas.
- No exageres los acentos rojos.
- No cambies el estilo de dibujo ni el grosor de líneas.

## 13. Gráficos, charts e infografías

Los gráficos presentan datos de forma atractiva y comprensible. Deben adaptarse al contexto y al formato.

### 13.1 Recursos obligatorios

Para gráficos básicos usa:

1. Tipografía oficial.
2. Paleta de colores secundarios.
3. Paleta de colores funcionales.

### 13.2 Tipos de gráficos

| Tipo | Uso |
|---|---|
| **Pie charts** | Círculo segmentado con descripciones conectadas por líneas de lectura, o comparación side by side. |
| **Bar charts verticales** | Permiten segmentar valores dentro de una misma barra. |
| **Bar charts horizontales** | Solo para mostrar valores únicos por barra. |
| **Line charts** | Líneas simples o gráficos mixtos con líneas y barras en el mismo eje. |

### 13.3 Dimensiones para folletos A4

| Tamaño | Ancho |
|---|---:|
| 1 columna | 81 mm |
| 2 columnas | 170 mm |

- Las barras y dimensiones de line charts o pie charts no deben superar **41 mm de altura**.

### 13.4 No hacer en charts

- No uses colores fuera de las paletas autorizadas.
- No uses tintes o gradaciones.
- No añadas líneas divisorias entre segmentos.
- No pongas diagramas sobre fondos de color.
- No reduzcas demasiado el espacio entre barras.
- No varíes grosores de barras dentro de un mismo gráfico.

## 14. Mapas

Los mapas combinan versiones estilizadas personalizadas y mapas generados por software. Deben comunicar un aspecto artesanal y premium.

### 14.1 Tipos de mapas

| Tipo | Uso |
|---|---|
| **World Map** | Presencia global y regiones donde opera la compañía. |
| **Location Map** | Redes sociales o folletos para demostrar expertise de mercado. |

### 14.2 World Map

- Debe aparecer siempre en gris.
- Se aplica como unidad completa.
- Puede funcionar como textura de fondo con un cuadro blanco superpuesto para información.
- Descargar en formato vectorial para cualquier tamaño.
- Tamaño mínimo: **200 mm**.
- Grosor de líneas: **3 pt**.
- Al escalar, trazos y formas deben escalar proporcionalmente.
- Para mapas del mundo en diseño de tiendas, contactar al equipo de Concepto y Diseño.

### 14.3 Location Map

#### Country Map

- Comunica datos regionales con precisión.
- Se divide en áreas según valores de información.
- Colores funcionales autorizados: gris, greige, amarillo, naranja, marrón, azul y verde.
- Segmentos con el mismo valor deben usar el mismo tono.
- Brand Red puede usarse únicamente como acento o destacado, no como gran área de color.

#### City, Region and Mountain Map

- Divide información en:
  - Datos de densidad.
  - Datos contextuales: agua, áreas verdes y carreteras.
- Colores autorizados:
  - Naranja y amarillo para densidad.
  - Verde, gris y azul para contexto.

### 14.4 Overlays

- Los overlays garantizan legibilidad y contraste sobre fondos detallados.
- No usar overlays en el mapa corporativo.

### 14.5 Software

- Para mapas corporativos y conjuntos de datos complejos se recomienda ArcGIS.

### 14.6 No hacer en mapas

- No colorees un mapa de país en rojo o naranja de forma dominante.
- No uses múltiples colores distintos para pintar un mapa de país sin lógica de datos.
- No apliques el mapa corporativo sobre fondos de color.
- No cambies los colores definidos para overlays.

## 16. Voz y tono

La voz de Engel & Völkers debe sentirse humana, segura, refinada y cercana. Las palabras se eligen cuidadosamente para conectar con las pasiones individuales de cada cliente y tratar a todas las personas como iguales.

### 16.1 Personalidad de escritura

- Selección meticulosa de palabras.
- Servicio personalizado.
- Relación equitativa con la audiencia.
- Comunicación significativa y capaz de captar atención.

### 16.2 Principios de voz

| Principio | Voz | Sí | No |
|---|---|---|---|
| **Attentive** | Respetuosa, cercana y adaptada a la audiencia. | Hablar directamente al lector; usar lenguaje conversacional y accesible. | Repetir palabras excesivamente; usar lenguaje demasiado formal o inaccesible. |
| **Discerning** | Flexible, refinada y optimizada para contexto. | Crear ritmo con recursos como aliteración; enfocarse en rasgos aspiracionales. | Usar lenguaje impersonal o tercera persona. |
| **Dynamic** | Con carácter, atractiva y capaz de mantener atención. | Estructura lineal clara; tono personal. | Oraciones complejas; lenguaje ornamentado o artificial. |
| **Confident** | Fuerte, decidida y clara. | Promesas con valor; lenguaje conciso e impactante. | Promesas tibias; fórmulas sin compromiso. |

### 16.3 Variación del tono

| Modo | Objetivo | Sí | No |
|---|---|---|---|
| **Informar / Inform** | Comunicar el mensaje clave de forma clara y accesible. | Datos y hechos relevantes; ir directo; oraciones simples. | Jerga, acrónimos o asumir conocimiento previo. |
| **Tranquilizar / Reassure** | Demostrar empatía y apoyo. | Explicar motivos con claridad; escribir de forma humana. | Maquillar malas noticias con optimismo excesivo; usar signos de exclamación. |
| **Alentar / Encourage** | Empoderar a la audiencia. | Usar “nosotros”; lenguaje conversacional. | Términos negativos o absolutos; exageraciones. |
| **Liderar / Lead** | Marcar un camino con seguridad. | Voz activa; datos; frases simples. | Atenuantes como “casi”, “normalmente”, “un poco”; sobrecomplicar. |
| **Inspirar / Inspire** | Crear emoción relevante y auténtica. | Ejemplos, adjetivos, ritmo, frases concisas y lenguaje positivo. | Ser aburrido o exagerar hasta sonar falso. |

### 16.4 Descripciones de propiedades

- Deben ser inspiradoras, claras y autoritativas.
- Deben vender la experiencia, no solo listar características.
- Deben ser concretas, sensoriales y elegantes.
- No deben sonar genéricas ni inconsistentes.
- Evita exageraciones vacías; usa detalles reales.

### 16.5 Reglas de copy para IA

Cuando escribas como Engel & Völkers:

- Habla directamente al lector.
- Usa frases claras y lineales.
- Sé cálido sin perder autoridad.
- Sé premium sin sonar distante.
- Usa datos cuando aporten confianza.
- Evita jerga, artificios y promesas vagas.
- Evita signos de exclamación en contextos sensibles.
- No uses un tono robótico.
- No uses lenguaje excesivamente ornamental.

## 18. Reglas maestras para IA generativa

Usa este bloque como instrucción base para una IA al crear diseños, textos o conceptos de Engel & Völkers:

```text
Crea contenido para Engel & Völkers con una estética premium, sobria, editorial y segura. Usa mucho espacio en blanco, composición con retícula, jerarquía clara, tipografía elegante y comunicación directa. Prioriza fotografía real y emocional antes que ilustración. Usa Brand Red solo como acento discreto, nunca como color dominante. Usa colores funcionales únicamente para mapas, charts e infografías. Mantén el logotipo sobre blanco o fondos de contraste aprobado, respeta áreas de seguridad y tamaños mínimos. Usa la villa como marca de calidad o cierre, nunca arriba ni sobre imágenes. No mezcles villa y línea roja en la misma composición. Escribe con voz atenta, analítica, dinámica y segura: humana, clara, refinada, directa y sin exageraciones. Evita layouts libres, exceso de color, jerga, iconos sobrecargados, ilustraciones como key visual y cualquier recurso que reduzca la sensación de lujo, precisión o confianza.
```

### Checklist rápido para evaluar una pieza

Antes de aprobar una pieza, verifica:

- [ ] ¿La composición usa retícula y márgenes claros?
- [ ] ¿Hay suficiente espacio en blanco?
- [ ] ¿El logo respeta tamaño, color y área de protección?
- [ ] ¿La villa se usa solo como cierre o marca de calidad?
- [ ] ¿El rojo aparece de forma contenida?
- [ ] ¿Los colores funcionales se usan solo en mapas/charts?
- [ ] ¿La tipografía respeta roles: Head para titulares, Text para cuerpo?
- [ ] ¿El contenido es claro, humano y seguro?
- [ ] ¿La fotografía tiene prioridad sobre ilustración?
- [ ] ¿Iconos, charts y mapas usan sus sistemas oficiales?
- [ ] ¿No se mezclan recursos incompatibles como villa + línea roja?
- [ ] ¿La pieza transmite sofisticación, precisión y confianza?

## 19. Prohibiciones críticas

Evita siempre:

- Usar rojo como color dominante.
- Usar colores fuera de paleta.
- Usar colores funcionales fuera de mapas/charts/infografías.
- Aplicar logo en color sobre fondos oscuros, imágenes o colores secundarios.
- Rehacer el logo o signet con tipografía común.
- Distorsionar, rotar, sombrear o modificar logos.
- Usar la villa sobre imágenes o fondos secundarios.
- Usar la villa en la parte superior.
- Combinar la villa con línea roja.
- Diseñar sin retícula.
- Usar patrones sobre fondos de color.
- Usar ilustraciones como sustituto de fotografía en portadas o key visuals.
- Mezclar ilustración y fotografía en una misma composición.
- Colorear iconos con paletas secundarias o funcionales.
- Usar charts con gradientes, tintes o colores no autorizados.
- Escribir textos largos, genéricos, impersonales o excesivamente ornamentados.
- Usar promesas vagas o tibias.
- Usar jerga innecesaria.
